# Classroom Engine Framework

## Core Principles

* Speed over intelligence
* Usability over features
* Teacher control over automation
* Fun first, learning second

## Classroom Constraints

* Must work in under 10 seconds
* Teacher has limited attention
* Students may be distracted
* System must reduce effort, not increase it

## UI Rules

* Large text (projector friendly)
* Minimal clutter
* Big buttons
* Dark mode preferred

## Data Model

* Use simple categories (Low/Mid/High)
* Avoid complex scoring systems

## Feature Rules

* Every feature must reduce teacher effort
* No unnecessary dashboards
* No overengineering

## AI Usage

* Use AI only for generating questions or prompts
* Do not use AI for control or decision-making

## Definition of Success

* Teacher can run a full activity with minimal effort
* Students are engaged quickly
* No friction during class
